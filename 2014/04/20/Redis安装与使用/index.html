PHP使用Reids

先安装php redis扩展，本人开发环境是windows，安装的时候需要注意版本，在php_info()的Compiler查看php是用什么编译成的，VC9还是VC6，然后从这里下载对应的版本，本机PHP Version 5.4.7，所以下在的是phpredis_5.4_vc9_ts.7z。

里面有两个文件，将php_igbinary.dll放到C:\Windows\System32下面，将php_redis.dll放到xampp\php\ext下面，不要忘了在php.ini中加入extension=php_redis.dll，好了，接下去重启apache，如果redis扩展和php版本不对应，apache在启动的时候会报错，一般情况下是版本错了。

CentOS下PHP Redis模块安装

下载并解压redis模块，下载地址

制作启动脚本
vim /etc/init.d/redis

设置开机启动
chmod 755 /etc/init.d/redis
chkconfig –add redis

按需求配置防火墙
-A INPUT -m state –state NEW -m tcp -p tcp –dport 6379 -j ACCEPT

启动redis
service redis start

查看日志
cat /data/logs/redis.log
