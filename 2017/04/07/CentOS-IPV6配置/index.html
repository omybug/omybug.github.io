iOS ipv6审核注意事项

确认一下几点：

  1. app server 是否支持。
  2. 第三方库是否支持（微信、微博、支付宝、环信等等）。
  3. 第三方服务是否支持，例如cdn、站点统计代码等等。
  4. 查看nginx是否支持ipv6 /usr/local/nginx/sbin/nginx -V 返回结果中，是否–with-ipv6
  5. 有可能是隧道节点的问题，可更换隧道节点尝试一下。
  6. 目前还遇到个情况，如果在空闲的情况下，ipv6会出现自动断开 解决办法，创建一个shell脚本，不停的连接ipv6网络，防止断开

因为我们App是H5外面套了壳，虽然登陆注册界面是使用原生开发的，但是审核了好几次都是因为登陆后，界面无法显示。环境都无法测试，最后发现是界面中的cdn资源，站点统计代码等链接无法访问，导致界面无法加载。
